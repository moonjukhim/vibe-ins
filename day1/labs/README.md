# DAY 1 · 기초 + 환경 (스포트라이트: 가입자 정리기 + 약관 추출기)

## M1. 자동차보험 가입자 명단 자동 정리기

**P1-1. 데이터 모델 만들기**
```
너는 Python을 가르치는 멘토야. 자동차보험 가입자 한 명을 표현하는 dataclass `Customer`를
만들어줘. 필드는 name(str), age(int), gender('M'/'F'), product(str), coverage(int, 보장금액 만원)야.
타입 힌트와 docstring을 넣고, 초보자가 읽기 쉽게 한 줄 주석을 달아줘. 코드만 깔끔하게.
```

**P1-2. 보험료 계산 함수**
```
`Customer`를 받아 월 보험료(만원)를 계산하는 `calc_premium(c)` 함수를 작성해줘.
규칙: 기본료는 상품별(건강보험 3·자동차보험 8·생명보험 5), 30세 초과 1세당 +2% 가중,
여성은 0.95 할인, 보장금액/1000에 비례. 결과는 소수 첫째 자리 반올림.
예) 42세 남성·자동차보험·5000만원 → 약 49.6. 함수 시그니처와 타입 힌트를 유지해.
```

**P1-3. CSV 읽기 + 예외 처리**
```
data/customers.csv를 읽어 Customer 리스트로 변환하는 `load_customers(path)`를 작성해줘.
제약: 숫자가 아닌 age나 빈 coverage처럼 깨진 행은 경고를 출력하고 건너뛰되(try/except),
전체 작업은 멈추지 않게 해. 인코딩은 utf-8-sig를 써. 어떤 예외를 잡는지 주석으로 설명해줘.
```

**P1-4. 연령대별 집계**
```
가입자 리스트를 받아 연령대(20대/30대/40대/50대+)별 평균 보험료를 dict로 반환하는
`summarize_by_age_group`를 작성해줘. 빈 그룹은 0으로 처리해 0으로 나누는 오류를 막아줘.
```

**P1-5. 검증 (필수)**
```
방금 만든 calc_premium에 대해 pytest 단위 테스트 5개를 작성해줘. 경계값(30세·50세 경계),
남/녀, 상품 3종을 각각 커버하고, 기대값을 직접 계산해 assert에 넣어줘.
```

**P1-6. 확장 — 재미 한 스푼**
```
calc_premium에 '무사고 할인'(무사고 연수 1년당 5%, 최대 30%) 로직을 추가하되,
기존 호출과 호환되도록 새 인자에 기본값을 둬. 변경 전후 동작 차이를 예시로 보여줘.
```

## M2. 환경 세팅 + 자동차보험 약관 조항 추출기

**P2-1. 실습 환경 점검**
```
내 PC가 실습 준비가 됐는지 점검하는 env_check.py를 작성해줘. Python 3.10 이상인지,
그리고 requests·bs4·pandas가 설치돼 있는지 확인해 각 항목을 [OK]/[FAIL]로 출력해줘.
```

**P2-2. (좋은 프롬프트 시연) LLM 약관 추출**
```
너는 보험 약관 분석 전문가야. 아래 자동차보험 약관 텍스트에서 4개 항목을 추출해 JSON으로 줘:
보장대상(배열), 면책사유(배열), 청구기한(문자열), 자기부담금(정수, 원).
한국어로 응답하고, 원문에 없는 값은 null로 표기해. 추측하지 마.
약관 텍스트:
"""
[가상 약관 텍스트 붙여넣기]
"""
```

**P2-3. 정규식 버전 추출기**
```
위 4개 항목을 정규표현식으로 추출하는 clause_extractor.py의 `extract_clauses(text)`를 작성해줘.
자기부담금은 쉼표 제거 후 정수로 변환하고, 못 찾은 항목은 None으로 둬. re.DOTALL을 사용하고
각 패턴에 무엇을 노리는지 주석을 달아줘.
```

**P2-4. 정규식 한계 점검 (검증)**
```
방금 만든 정규식이 놓칠 수 있는 약관 표현 5가지를 예시로 보여주고, 각각을 잡기 위한
보완 패턴을 제안해줘. 왜 기존 패턴이 실패하는지도 한 줄로 설명해줘.
```

**P2-5. 정규식 → LLM 폴백 파이프라인**
```
정규식으로 추출하되, None이 나온 항목만 LLM에 다시 물어 채우는 구조로 바꿔줘.
폴백 호출 전후에 로그를 남기고, 최종 결과를 output/clauses.json으로 저장해줘.
```

**P2-6. PDF 약관 대응 (확장)**
```
약관이 PDF로 올 경우 pdfplumber로 텍스트를 추출해 extract_clauses에 넘기는 코드를 추가해줘.
표가 포함된 페이지도 텍스트가 깨지지 않게 처리 방법을 알려줘.
```
